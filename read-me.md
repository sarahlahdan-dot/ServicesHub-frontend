## Brief Frontend Overview

This frontend is a React and Vite web application for a local services marketplace. It allows users to browse available services, search by category or keyword, open service details, and create an account or sign in.

The app supports different user roles such as customer, provider, and admin. Customers can add services to a cart, book services, send messages to providers, and leave reviews. Providers can manage their own services and bookings from the dashboard, while admins have access to an admin dashboard.

The frontend connects to the backend with Axios APIs and uses React Router for page navigation. Overall, the project focuses on making service discovery, booking, communication, and account management simple and clear for users.
